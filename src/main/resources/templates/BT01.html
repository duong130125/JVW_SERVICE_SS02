Java Persistence API (JPA) là một đặc tả (specification) trong Java dùng để quản lý dữ liệu giữa các đối tượng Java và cơ sở dữ liệu quan hệ. JPA không phải là một framework cụ thể mà là một tập hợp các quy tắc, còn các framework như Hibernate là hiện thực (implementation) của JPA. JPA giúp lập trình viên thao tác với cơ sở dữ liệu thông qua các đối tượng thay vì truy vấn SQL thuần túy, từ đó tăng tính trừu tượng và giảm phụ thuộc vào cơ sở dữ liệu cụ thể. Hibernate là một trong những hiện thực phổ biến nhất của JPA, cung cấp nhiều tính năng mở rộng như caching và lazy loading. JPA đóng vai trò quan trọng trong phát triển ứng dụng Java vì nó chuẩn hóa cách truy cập dữ liệu, dễ tích hợp với các framework như Spring và giúp ứng dụng dễ bảo trì, mở rộng hơn.
